Chickadee know what you hear in nature.

www.chickadee.io GITHUB
  • FEATURES:
  • Backbone client-side consuming RESTful Rails API
  • responsive design and layout with jQuery & CSS
  • asynchronous UI management during AJAX requests
  • media storage with Amazon Web Service's S3
  • Omniauth with Facebook

Chickadee is a quiz application for learning to identify birds by their songs. It harnesses bird demographic data collected by citizen scientists from the Cornell Laboratory of Ornithology's eBird database, as well as audio recordings of relevant species from Xeno-canto's library. Chickadee allows users to custom-tailor their experience, studying birds from their regions of choice. Check it out!
